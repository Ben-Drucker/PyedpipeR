Convert default arguments to string format.

Parameters
default_args:list[convertable_types]The default arguments to convert. Can be either a "singleton" (i.e. a single value) or a list of values.
Returns
Sequence[convertable_types]If returning a singleton, returns the converted default argument in string format. If returning a list, returns a list of converted default arguments in string format.
